## 상세 분석

### 📍 Problem (문제)

**소규모 모임 주최자의 페인 포인트:**

| 문제                         | 현황                                 | 영향                         |
| ---------------------------- | ------------------------------------ | ---------------------------- |
| **산발적 정보 관리**         | 엑셀, 카톡, 구글폼 등 여러 도구 사용 | 정보 중복, 최신성 보장 안 함 |
| **참여 인원 파악 어려움**    | 수락/거절/미답 상태가 섞임           | 모임 준비 시 인원 불확실     |
| **모임 당일 정보 공유 부재** | "어디서 만나?" 반복 질문             | 참여자 혼란, 시간 낭비       |
| **카풀 수동 매칭**           | 개별적으로 연락하여 협의             | 주최자 부담 증가, 오류 발생  |
| **복잡한 정산 계산**         | 누가 뭘 안 냈는지 추적 어려움        | 모임 후 갈등 발생 가능       |
| **연락처 산재**              | 카톡, 전화번호 등 여러 곳 존재       | 다음 모임 연락 어려움        |

**타겟 사용자가 겪는 반복적 문제:**

- 월 1-4회 소규모 모임(5-20명) 주최자
- 수영장, 헬스장, 친구 그룹, 동호회 등
- 현재 도구: Excel, Google Sheets, Kakao Talk, SMS

### 💡 Solution (해결책)

**5가지 핵심 기능으로 모임 전 과정 통합:**

#### F001: 이벤트 생성/관리

```
주최자가 한 곳에서 이벤트 정보를 입력하면,
공개 링크 자동 생성 → 카톡/문자로 공유 가능
```

- 제목, 날짜/시간, 장소, 인원 제한, 승인 방식 설정
- 이벤트별 고유 공유 링크 생성
- 언제든 수정/취소 가능

#### F002: 참여자 관리

```
공개 링크로 참여 신청 → 주최자 승인/거절
정원 초과 시 자동으로 대기자 관리
```

- 신청, 승인, 거절, 대기자 상태 관리
- 참여자 목록 한 눈에 파악
- 단체 승인/거절 기능 (선택)

#### F003: 공지 관리

```
모임 전/후 필요한 정보를 중앙 집중식으로 공유
"모이는 시간", "정확한 위치", "준비물" 등
```

- 중요 공지 상단 고정
- 시간순 공지 목록
- 참여자는 읽기만 가능

#### F004: 카풀 매칭

```
운전자 등록 + 탑승 희망자 등록 → 같은 지역끼리 자동 연결
```

- 출발 지역별 운전자/탑승자 그룹화
- 좌석 관리 (운전자가 가능 좌석 수 입력)
- 주최자가 매칭 처리

#### F005: 1/N 정산

```
비용 입력 → 참여자 수로 자동 균등 분배
누가 얼마를 내야 하는지 명확함
```

- 비용 항목별 입력 (식사비, 교통비 등)
- 참여자 수 기준 자동 계산
- 개인별 납부 현황 추적
- 납부 완료 체크

### 🚀 Unfair Advantage (부당한 이점)

**복사하기 어려운 경쟁 우위:**

1. **소규모 모임 최적화**
   - 복잡도 ↓, 사용성 ↑ (5-20명 기준)
   - 큰 플랫폼(Eventbrite 등)의 과도한 기능 제거

2. **카풀 자동 매칭 엔진**
   - 지역 기반 자동 그룹핑
   - 좌석 수 관리로 정확성 보장
   - 대규모 이벤트 플랫폼과 다른 초점

3. **1/N 자동 계산**
   - 정산 분쟁 해결 (명확한 금액)
   - 참여자 수 변동 시 자동 재계산

4. **초기 고객층의 긍정적 평판**
   - 소규모 모임 커뮤니티 내 입소문
   - 정기 모임 → 반복 사용자 확보
   - 사용자 피드백으로 빠른 개선

5. **Niche Market Focus**
   - 대규모 이벤트 플랫폼과 다른 타겟
   - 경쟁사 관심도 ↓

### 🎯 Customer Segments (고객 세그먼트)

**1단계: 조기 고객 (Early Adopters)**

- 소규모 모임(5-20명) 정기 주최자
- 현재 도구의 불편함을 체감하는 그룹
- 초대 링크 공유로 쉽게 확산

**타겟 그룹:**

- 🏊 **수영/헬스 클럽**: 정기 모임
- 👥 **친구 그룹**: 정기 여행, 야외활동
- 🎯 **소모임/동호회**: 정기 모임 관리 필요
- 🏘️ **아파트 주민회**: 행사 관리
- 🚗 **카풀 커뮤니티**: 카풀 매칭 기능

**2단계: 성숙 고객 (Growth Phase)**

- 정기적으로 많은 수의 모임을 주최하는 조직
- 커뮤니티 리더, 단체 여행사 등
- 프리미엄 기능 ($) 수용 가능

### 💰 Revenue Streams (수익원)

**3단계 수익화 모델:**

#### Phase 1: Free (초기)

- **모든 기본 기능 무료**
  - 이벤트 생성 (무제한)
  - 참여자 관리
  - 공지 관리
  - 카풀 매칭
  - 1/N 정산
- **목표**: 사용자 확보, 피드백 수집

#### Phase 2: Freemium ($4.99-9.99/월)

- **프리미엄 기능**
  - 실시간 알림 (Slack, Email, Push)
  - 결제 연동 (Toss, Stripe)
  - 고급 분석 (참여 현황 차트)
  - 대용량 파일 첨부
  - 우선 고객 지원

#### Phase 3: B2B ($49/월)

- **조직용 패키지**
  - 복수 이벤트 관리 권한
  - 팀 멤버 추가
  - 커스텀 브랜딩

**초기 수익 모델**: Free (입소문) → Freemium (지속 가능성)

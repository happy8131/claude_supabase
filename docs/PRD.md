# 모임 이벤트 관리 MVP PRD

## 🎯 핵심 정보

**목적**: 모임 주최자가 공지·참여자 관리·카풀·정산을 하나의 웹에서 처리할 수 있게 한다
**사용자**: 수영/헬스/친구 모임을 정기적으로 주관하는 소규모 그룹 주최자와 참여자

---

## 🚶 사용자 여정

```
👤 주최자 플로우

1. 랜딩 페이지
   ↓ 로그인/회원가입 버튼 클릭

2. 로그인 페이지 → 인증 완료
   ↓ 자동 리다이렉트

3. 대시보드 (내 이벤트 목록)
   ↓ "새 모임 만들기" 버튼 클릭

4. 이벤트 생성 페이지
   ↓ 저장 완료

5. 이벤트 허브
   ↓ "공유 링크 복사" → 카카오/문자 공유

   [참여 신청 들어오면] → 참여자 관리 탭 → 승인/거절
   [모임 당일] → 카풀 탭 → 매칭 현황 확인
   [모임 후] → 정산 탭 → 비용 입력 → 1/N 계산 → 납부 체크

---

👤 참여자 플로우

1. 공유 링크 클릭 → 이벤트 공개 상세 페이지
   ↓ "참여 신청" 버튼 클릭

   [비로그인] → 로그인 페이지 (next 파라미터 포함) → 로그인 후 자동 신청
   [로그인 상태] → 즉시 참여 신청

2. 신청 결과 확인
   [자동 승인 이벤트] → 바로 이벤트 허브 진입
   [승인 필요 이벤트] → "승인 대기 중" 상태 → 주최자 승인 후 이벤트 허브 진입

3. 이벤트 허브 (참여자 뷰)
   ↓ 공지 탭: 주최자 공지 확인
   ↓ 카풀 탭: 운전자 등록 또는 탑승 희망 등록
   ↓ 정산 탭: 1인당 금액 확인, 납부 완료 체크
```

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID       | 기능명           | 설명                                            | MVP 필수 이유                  | 관련 페이지                             |
| -------- | ---------------- | ----------------------------------------------- | ------------------------------ | --------------------------------------- |
| **F001** | 이벤트 생성/관리 | 모임 이벤트 생성, 수정, 삭제 및 공유 링크 발급  | 서비스의 핵심 단위             | 이벤트 생성 페이지, 이벤트 허브         |
| **F002** | 참여자 관리      | 공개 참여 신청 수락, 승인/거절, 정원 관리       | 모임 인원 파악의 핵심          | 이벤트 공개 상세 페이지, 참여자 관리 탭 |
| **F003** | 공지 관리        | 공지 작성/수정/삭제 및 중요 공지 고정           | 참여자와의 소통 채널           | 공지 탭                                 |
| **F004** | 카풀 매칭        | 출발 지역 기반 운전자↔탑승자 그룹 매칭          | 오프라인 모임의 핵심 이동 수단 | 카풀 탭                                 |
| **F005** | 1/N 정산         | 비용 항목 입력 후 참여자 수 기준 자동 균등 분배 | 모임 후 정산 분쟁 방지         | 정산 탭                                 |

### 2. MVP 필수 지원 기능

| ID       | 기능명      | 설명                                         | MVP 필수 이유                  | 관련 페이지                    |
| -------- | ----------- | -------------------------------------------- | ------------------------------ | ------------------------------ |
| **F010** | 기본 인증   | 이메일/Google OAuth 회원가입·로그인·로그아웃 | 서비스 이용을 위한 최소 인증   | 로그인 페이지, 회원가입 페이지 |
| **F011** | 기본 프로필 | 이름, 사용자명 관리                          | 참여자 식별에 필요한 최소 정보 | 프로필 페이지                  |

### 3. MVP 이후 기능 (제외)

- 실시간 알림 (이메일/앱 푸시)
- 지도 연동 카풀 (현재는 텍스트 지역명 매칭)
- 정산 외부 연동 (토스페이 등)
- 아바타·소개 등 상세 프로필
- 이벤트 반복 일정 (정기 모임)

| **사용자 행동** | 서비스 기능 확인 후 회원가입 또는 로그인                                 |
| **주요 기능**   | • 서비스 핵심 기능 소개 카드<br>• **로그인** 버튼<br>• **회원가입** 버튼 |
| **다음 이동**   | 로그인 클릭 → 로그인 페이지, 회원가입 클릭 → 회원가입 페이지             |

| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 공유 링크로 접근한 비회원/회원에게 이벤트 소개 및 참여 신청 제공                                                                                                           |

| **사용자 행동** | 이벤트 내용 확인 후 참여 신청                                                                                                                                              |
| **주요 기능**   | • 이벤트 제목/날짜/장소/설명 표시<br>• 현재 참여 인원 / 최대 인원 표시<br>• 참여 상태별 버튼 분기 (신청 가능 / 승인 대기 중 / 참여 중 / 정원 마감)<br>• **참여 신청** 버튼 |
| **다음 이동**   | 비로그인 신청 → 로그인 페이지 (`?next=` 포함), 신청 완료 → 상태 버튼 변경                                                                                                  |

> **구현 기능:** `F004` | **메뉴 위치:** 이벤트 허브 > 카풀 탭

| 항목            | 내용                                                                                                                                                                                                                                            |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 출발 지역 기반으로 운전자와 탑승자를 매칭                                                                                                                                                                                                       |
